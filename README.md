# wepy-echart
echart官方提供了小程序版本，但使用的是原生语言，此项目示例了仪表盘的用法，其他图形修改配置即可。wepy安装过程已省略

# 使用步骤
1. 将lib目录拷贝至工程目录下

2.将components下的 echart.wpy 目录拷贝至你的工程目录中

3.在index.wpy中使用定义的组件
